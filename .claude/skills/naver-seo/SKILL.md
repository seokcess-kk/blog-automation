# Naver SEO Guidelines

> **enforcement: suggest**
> 네이버 블로그 SEO 최적화 규칙

---

## 1. 네이버 검색 알고리즘 이해

### 1.1 C-Rank (Creator Rank)

창작자 신뢰도 기반 랭킹:
- **블로그 활동 이력**: 운영 기간, 발행 빈도
- **원본 콘텐츠 비율**: 복붙 vs 창작
- **체류 시간**: 글 품질 지표
- **이웃/팬 반응**: 좋아요, 댓글, 공감

### 1.2 D.I.A (Deep Intent Analysis)

검색 의도 분석:
- **정보성 키워드**: 상세한 정보 콘텐츠 선호
- **상업성 키워드**: 비교/리뷰 콘텐츠 선호
- **지역 키워드**: 위치 정보 포함 콘텐츠 선호

## 2. 제목 최적화

### 2.1 제목 작성 규칙

```
[키워드 포함] + [구체적 정보] + [클릭 유도]
```

| 패턴 | 예시 | 효과 |
|------|------|------|
| 숫자 활용 | "강남 피부과 추천 TOP 5" | 구체성 ↑ |
| 질문형 | "여드름 치료 어디서?" | 검색 의도 매칭 |
| 연도 포함 | "2025 다이어트 한의원 비교" | 최신성 ↑ |
| 지역 포함 | "홍대 맛집 현지인 추천" | 지역 검색 매칭 |

### 2.2 제목 길이

- **권장**: 20~40자
- **최대**: 60자 (초과 시 잘림)
- **키워드 위치**: 앞쪽 1/3 내 배치

### 2.3 금지 패턴

```
# 네이버 저품질 탐지 패턴
- 키워드 반복: "강남피부과 강남 피부과 피부과강남"
- 특수문자 남용: "★★★강남피부과★★★"
- 낚시성 제목: "충격! 이것만 알면..."
```

## 3. 본문 최적화

### 3.1 글자수 가이드

| 유형 | 권장 글자수 | 비고 |
|------|-------------|------|
| 정보성 | 2,000~4,000자 | 상세할수록 유리 |
| 리뷰 | 1,500~3,000자 | 경험 중심 |
| 지역정보 | 1,000~2,000자 | 핵심 정보 위주 |

### 3.2 키워드 배치

```
[제목] 메인 키워드 필수
[첫 문단] 메인 키워드 1회
[소제목] 연관 키워드 배치
[본문] 자연스러운 키워드 분산 (밀도 1~2%)
[마지막 문단] 메인 키워드 1회
```

### 3.3 소제목 구조

```markdown
## 1. 메인 소제목 (키워드 포함)
본문 텍스트...

## 2. 서브 소제목 (연관 키워드)
본문 텍스트...

## 3. 정리/결론
본문 텍스트...
```

- **소제목 수**: 3~7개 권장
- **소제목 길이**: 10~25자
- **키워드 포함율**: 소제목의 50% 이상

### 3.4 문단 구조

- **문단 길이**: 3~5문장 (모바일 가독성)
- **줄바꿈**: 문단 간 1줄 공백
- **핵심 강조**: 굵게/색상 (과용 금지)

## 4. 이미지 최적화

### 4.1 이미지 수량

| 글자수 | 권장 이미지 수 |
|--------|----------------|
| 1,000자 이하 | 2~3장 |
| 1,000~2,000자 | 3~5장 |
| 2,000자 이상 | 5~10장 |

### 4.2 이미지 규격

- **크기**: 가로 최소 800px (권장 1200px)
- **비율**: 16:9 또는 4:3
- **용량**: 1MB 이하 (로딩 속도)
- **형식**: JPEG 권장

### 4.3 EXIF 메타데이터

```python
# 카메라 정보 삽입 (AI 이미지 위장)
exif_data = {
    "Make": random.choice(["Apple", "Samsung"]),
    "Model": random.choice(["iPhone 15 Pro", "Galaxy S24"]),
    "DateTime": generate_recent_date(days=7),
    "GPSInfo": None,  # GPS 제거 (프라이버시)
}
```

### 4.4 대체 텍스트

```
# 이미지 대체 텍스트 패턴
[키워드] + [이미지 설명]

예: "강남 피부과 내부 인테리어"
예: "여드름 치료 전후 비교"
```

## 5. 태그 최적화

### 5.1 태그 수

- **권장**: 5~10개
- **최대**: 15개

### 5.2 태그 구성

```
1. 메인 키워드: "강남피부과"
2. 변형 키워드: "강남 피부과", "강남피부과추천"
3. 연관 키워드: "여드름치료", "피부관리"
4. 지역 태그: "강남", "신논현", "강남역"
5. 시즌/트렌드: "2025피부과", "봄철피부관리"
```

### 5.3 태그 주의사항

- 본문에 없는 키워드 태그 금지 (스팸 탐지)
- 경쟁사 브랜드명 태그 금지

## 6. 발행 타이밍

### 6.1 최적 발행 시간

| 요일 | 권장 시간 | 비고 |
|------|-----------|------|
| 평일 | 09:00~11:00 | 출근 후 검색 피크 |
| 평일 | 13:00~14:00 | 점심시간 |
| 평일 | 17:00~18:00 | 퇴근 전 |
| 주말 | 10:00~12:00 | 오전 여유 시간 |

### 6.2 발행 빈도

- **신규 블로그**: 주 3~5회 (안정화까지)
- **안정화 블로그**: 주 2~3회
- **의료 키워드**: 주 1~2회 (conservative)

---

## 7. 상위노출 체크리스트

### 7.1 발행 전 체크

```
[ ] 제목에 메인 키워드 포함
[ ] 제목 40자 이내
[ ] 본문 첫 문단에 키워드 포함
[ ] 소제목 3개 이상
[ ] 이미지 3장 이상 + EXIF 삽입
[ ] 글자수 1,500자 이상
[ ] 태그 5개 이상
[ ] 의료광고법 검증 완료
```

### 7.2 발행 후 체크

```
[ ] 24시간 내 순위 확인
[ ] 7일 후 순위 변동 모니터링
[ ] 이웃 반응 (좋아요, 댓글) 체크
[ ] 저품질 여부 확인
```

## 8. 저품질 방지

### 8.1 저품질 탐지 요소

```
# 콘텐츠 품질
- 복붙 콘텐츠 (유사도 높은 글)
- 키워드 스터핑 (과도한 키워드 반복)
- 의미 없는 글 (AI 감지)

# 행동 패턴
- 대량 발행 (하루 5개 초과)
- 비정상 시간 발행 (새벽 2~5시)
- 패턴화된 발행 (매일 정각)

# 외부 요소
- 비정상 트래픽 유입
- 자동화 봇 탐지
```

### 8.2 저품질 복구

1. 문제 글 비공개 처리
2. 2~4주 정상 운영
3. 고품질 콘텐츠 발행 (주 2회)
4. 이웃 소통 활성화
